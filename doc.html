    GET /get_block_by_number

    Description

    Retrieve a specific block by its number. If a valid query-param api_key is provided, transactions are included. Otherwise, they are omitted.

    Query params

  • network: str
  • block_number: int
  • api_key: str (optional)

    • Response

    • block: dict
      • block_number: int
      • consensus_root: str
      • hash: str
      • previous_hash: str
      • timestamp: float
      • transactions: dict (optional)
        • hash: str
        • nonce: str
        • post_timestamp: str
        • public_key: str
        • sender: str
        • signature: str
        • timestamp: str
        • vector: list
        • vector_signature: str
      • validator: str
    • network: str

    Example

    https://api.sentichain.com/blockchain/get_block_by_number?network=mainnet&block_number=1
    https://api.sentichain.com/blockchain/get_block_by_number?network=mainnet&block_number=1&api_key=abc123

    GET /get_points_by_block_range_no_embedding

    Description

    Return snapshots of Event Map for blocks in [start_block, end_block]. If a valid query-param api_key is provided, contents are included. Otherwise, they are hidden.

    Query params

  • start_block: int
  • end_block: int
  • api_key: str (optional)

    • Response

    • points: list
      • block_number: int
      • post_link: str
      • post_content: str
      • point_x: float
      • point_y: float
      • cluster_number: int
      • cluster_x: float
      • cluster_y: float
      • cluster_summary_short: str
      • cluster_summary_long: str

    Example

    https://api.sentichain.com/mapper/get_points_by_block_range_no_embedding?start_block=1&end_block=10
    https://api.sentichain.com/mapper/get_points_by_block_range_no_embedding?start_block=1&end_block=10&api_key=abc123
