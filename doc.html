    GET /get_last_block

    Description

    Retrieve the latest block. If a valid query-param api_key is provided, full transactions are included. Otherwise, they are hidden.

    Cost

    Each request will cost 1 point from the balance of the API key.

    Query params

  • network: str
  • api_key: str (optional)

    • Response

    • block: dict
      • block_number: int
      • consensus_root: str
      • hash: str
      • previous_hash: str
      • timestamp: float
      • transactions: dict
        • hash: str
        • nonce: str
        • post_timestamp: str
        • post_content: str ("hidden" if no valid API key)
        • post_link: str ("hidden" if no valid API key)
        • public_key: str
        • sender: str
        • signature: str
        • timestamp: str
        • vector: list ("hidden" if no valid API key)
        • vector_signature: str
      • validator: str
    • network: str

    Example

    https://api.sentichain.com/blockchain/get_last_block?network=mainnet
    https://api.sentichain.com/blockchain/get_last_block?network=mainnet&api_key=abc123

    GET /get_reasoning_match_chunk_end

    Description

    Return the reasoning for the last available chunk whose chunk_end is less and equal to user_chunk_end, for the given ticker and summary_type. If a valid query-param api_key is provided, reasoning is included. Otherwise, it is hidden. Please note, api_key is not required for summary_type ending with _public.

    Cost

    Each request will cost 1 point from the balance of the API key.

    Query params

  • ticker: str
  • summary_type: str
  • user_chunk_end: int
  • api_key: str (optional)

    • Response

  • reasoning: str

    • Example

    https://api.sentichain.com/agent/get_reasoning_match_chunk_end?ticker=DOGE&summary_type=observation_public&user_chunk_end=200
    https://api.sentichain.com/agent/get_reasoning_match_chunk_end?ticker=DOGE&summary_type=observation_public&user_chunk_end=200&api_key=abc123
